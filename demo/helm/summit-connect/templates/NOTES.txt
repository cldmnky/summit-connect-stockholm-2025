Summit Connect Virtual Machine has been deployed!

To check the VM status:
  kubectl get vm {{ .Values.vm.name | default (include "summit-connect.fullname" .) }} -n {{ .Values.vm.namespace | default .Release.Namespace }}

To get the VM instance status:
  kubectl get vmi {{ .Values.vm.name | default (include "summit-connect.fullname" .) }} -n {{ .Values.vm.namespace | default .Release.Namespace }}

To access the VM via console:
  virtctl console {{ .Values.vm.name | default (include "summit-connect.fullname" .) }} -n {{ .Values.vm.namespace | default .Release.Namespace }}

To SSH into the VM (once it's running):
  virtctl ssh rhel@{{ .Values.vm.name | default (include "summit-connect.fullname" .) }} -n {{ .Values.vm.namespace | default .Release.Namespace }}
